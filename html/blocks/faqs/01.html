Frequently Asked Questions

What is UnoUI?

UnoUI is a modern UI collection offering ready-to-use components, blocks, and templates to help you build beautiful web interfaces faster.

Can I use UnoUI with any framework?

Yes, UnoUI is framework-agnostic and can be integrated with any modern frontend stack, including React, Vue, Angular, and plain HTML/CSS.

Are UnoUI components customizable?

Absolutely! All components are fully customizable with utility classes and variables, so you can match your brand and design system.

Is UnoUI open source?

Yes, UnoUI is open source and free to use for personal and commercial projects. Contributions are welcome!

How do I get started with UnoUI?

Simply browse the UnoUI documentation, copy the code snippets, and start building. No installation required for basic usage.
